CloakFrame 1.11.0 upgrades the built-in face models, replaces the fixed smiley
sticker with custom image overlays, and lets video review add missed regions.

### New face models
- Uses YOLO5Face-n as the recommended accurate model for photos and videos
- Adds YuNet as a lightweight CPU model with a download size under 0.3 MB
- Keeps custom SCRFD ONNX files available through Browse

### Custom image overlays
- Select a PNG, JPEG, BMP, TIFF, or WebP image to cover detected faces and
  license plates in photos, videos, style previews, and image review
- Keeps the custom image's aspect ratio and aligns face overlays to the tilt
  reported by landmark-capable face models
- Interpolates and smooths overlay position, size, and tilt along video tracks
- Preserves transparency while applying a safety mosaic underneath, so
  transparent pixels never reveal the original detected region
- Uses premultiplied-alpha transforms to avoid dark fringes around rotated
  transparent images

### Video review
- Draw missed regions as manual tracks before encoding
- Limit each manual track to the relevant time range
- Add keyframes as a region moves and interpolate the boxes between them
- Remove manual tracks or continue excluding false automatic tracks

### Reliability and migration
- Preserves the previous Fast or Accurate preference when migrating existing
  settings to the new model choices
- Runs YOLO5Face-n through ONNX Runtime with GPU acceleration and automatic CPU
  fallback, while YuNet uses OpenCV's CPU backend
- Bounds model download size and verifies the SHA-256 digest, input shape, and
  output shape before processing
- Pins the YuNet download to a specific OpenCV Zoo revision for reproducible
  model retrieval
- Loads a bounded custom-image snapshot before each run so later file changes
  cannot alter an active photo or video job
- Supports custom-image coverage for 8-bit and 16-bit photos and the parallel
  video masking pipeline

The face models are downloaded on first use and then work offline. YOLO5Face-n
is based on WIDER FACE training data and should be treated as non-commercial
only. YuNet is MIT-licensed. Existing custom SCRFD models keep their own
upstream license terms.
